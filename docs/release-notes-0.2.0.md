# Discord Media Exporter 0.2.0

限定配布向けの最初の検証版です。

## 主な機能

- 現在のメッセージ表示領域に見えているDiscord添付を一覧化
- 画像、動画、その他添付の絞り込みと選択
- 安全なファイル名で最大3件ずつ保存
- ファイル単位の進捗と失敗理由の表示
- popupやservice workerの再起動後の進捗復元

## 権限

- `activeTab`: ユーザーが拡張機能を開いたタブを一時的に確認
- `scripting`: そのタブへ表示範囲の確認処理を注入
- `downloads`: 選択したファイルの保存と進捗確認
- `storage`: 検証済み候補と進捗をブラウザセッション内で保持

恒久的host permission、Cookie、ユーザートークン、Discord内部APIは使用しません。

## 既知の制約

- 自動更新はありません
- 画面外や過去ログを自動取得しません
- DiscordのDOM変更により検出できなくなる場合があります
- スポイラー付き添付など一部の表示は未検証です

## 更新時の注意

[インストール・更新ガイド](installation.md)に従ってSHA-256を照合してください。権限が上記より増えている場合はインストールせず、配布元へ確認してください。
