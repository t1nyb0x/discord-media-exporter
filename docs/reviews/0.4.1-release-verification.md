## 修正対象

- `0.4.0`で、自動収集が継続中でも時間を置いてpopupを再表示するとボタンがOFF状態になる問題

## 修正内容

- collector状態と停止応答を、認識済み要求にだけ`Promise<CollectorResponse>`で返す
- popup再表示時のON復元テストを追加
- collector entrypointがPromise responseを返すテストを追加
