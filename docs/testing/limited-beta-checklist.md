# 限定配布テストチェックリスト

## テスト前

- [ ] テスター本人が参加に同意している
- [ ] 権利が明確なテスト用Discordチャンネルを使う
- [ ] 配布ZIPのSHA-256が一致している
- [ ] リリースノートのバージョンとChrome表示のバージョンが一致している
- [ ] 権限が`activeTab`, `scripting`, `downloads`, `storage`だけである

## 基本操作

- [ ] 表示中の画像、動画、その他添付が一覧に出る
- [ ] 画面外の添付が一覧に出ない
- [ ] 種類フィルター、全選択、解除をキーボードでも操作できる
- [ ] 初期状態では全件未選択である
- [ ] 選択したファイルだけが保存される
- [ ] ファイル単位の待機、保存中、完了が表示される
- [ ] 4件以上で最大3件ずつ処理される

## 安全な失敗

- [ ] Discord以外のページではスキャンしない
- [ ] 対応できないDiscord画面では解析失敗または対象外を表示する
- [ ] 一件が失敗しても他の候補が継続する
- [ ] エラー表示に完全なURL、チャンネル名、ユーザー名が出ない
- [ ] popupを閉じて再度開いても進捗を確認できる

## フィードバック

次だけを報告し、メッセージ本文、ユーザー名、チャンネル名、完全なURL、ダウンロード先パスは送らないでください。

- 拡張機能バージョン
- ChromeバージョンとOS
- 操作手順
- 期待した候補件数と実際の候補件数
- URLを含まないエラー文
- 再現頻度

## テスト後

- [ ] 不要なテストファイルを削除した
- [ ] 問題がある場合は拡張機能を無効化した
- [ ] 継続利用しない場合は拡張機能と展開ディレクトリを削除した
