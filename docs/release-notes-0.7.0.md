# Discord Media Exporter 0.7.0 リリースノート

## 概要

popupの候補一覧で、画像添付の内容を保存前に確認できるサムネイル表示を追加しました。

動画とその他ファイル、サムネイルを取得できない画像では、従来の種類アイコンを表示します。

## 追加

- 検証済みDiscord画像添付の80×80サムネイル
- popup内でのサムネイル遅延読み込みと非同期decode
- 読み込み完了までは種類アイコンを表示するフォールバック
- 読み込み失敗時にサムネイルを除去して種類アイコンへ戻す処理

## セキュリティとプライバシー

- サムネイルURLを既存allowlistで検証
- 画像通信先をDiscord media proxyの`media.discordapp.net`へ限定
- extension pageのCSPでも画像通信先を同じホストへ限定
- 恒久的host permission、新しい外部依存、外部サーバーを追加しない
- サムネイルを永続保存しない
- 動画本体をプレビュー目的で読み込まない

## 維持する境界

- ユーザーが表示した範囲から取得済みの候補だけを一覧表示
- Discord内部API、token、Cookieを不使用
- 画面外メディアの新規収集、期限切れURLや権限の回避を不使用
- 個別保存とZIP保存の既存動作を維持

## 検証

- lint
- TypeScript型検査
- 単体テスト
- Prettier整形チェック
- production build
- 配布ZIP、manifest、SHA-256のrelease検証

Project ownerは2026-07-17にChrome実機でサムネイル表示を確認し、問題がないことを報告しました。ChromeバージョンとOSは共有されていないため推測で補完しません。

## 既知の制約

- Discord CDN URLの有効期限切れや取得失敗時はサムネイルを表示できない
- 動画とその他ファイルは種類アイコンを表示する
